# Zerg Runtime

> ~ build the minimal runtime for zerglang ~

This is the minimal runtime for Zerg that provides the smallest set of the core functions required to
run Zerg programs, and use it as the bootstrap for building the full-featured runtime. The runtime is
designed to be lightweight and efficient, making it suitable for any environment where resources are
limited, and designed to be compiled as the intepreter and used to run Zerg programs or build the
full-featured runtime.

The runtime is implemented based on the Go programming language as the Zerg intepreter, and it is
designed to build and compile Zerg programs. It provides the subset of the Zerg language GRAMMAR that
is necessary to run and build the runtime itself.

## Features
The minimal runtime in Zerg includes the necessary features to run and build the runtime that writes
by itself. It includes the following features:

- [ ] Basic I/O operations
- [ ] Memory management
- [ ] Basic data types and structures
- [ ] Control flow constructs
- [ ] Error handling mechanisms
