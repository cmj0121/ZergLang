# Zerg #
![Build Status](https://github.com/cmj0121/ZergLang/workflows/test/badge.svg)

The Zerg is a compiled language (ANSI C based version), and has a sub-project called Zasm which
is the assembler used on the Zerg. This project is developed via the clang but expected the code
is compatible with other [ANSI C][1]/C11 compile.

## Roadmap ##
- [ ] lexer and parser
- [ ] assembler
	- [ ] define grammar
	- [ ] code emitter / x86-64
- [ ] compiler
	- [ ] define grammar

[1]: https://en.wikipedia.org/wiki/ANSI_C
