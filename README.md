# Zerg

> Write your programs as zerg-rush

Zerg is the open source programming language for you to write your programs as zerg-rush.

It help you to easyly write your ideas into programs, with simple and clear syntax without complicated rules.
You can focus on your ideas and creativity, instead of wasting time on learning complex programming languages,
and you can just write the code as you think, with one way and should be only one way to do it.

The Zerg language is designed to follow principles of:

- small and crisp
- procedural
- concurrent
- garbage-collected
- strongly-typed language

## DDD (Dream-Driven Development)

This project is based on the DDD (dream-driven development) methodology which means
the project is based on what I dream of.

All the features are based on my needs and my dreams.
